# Data Visualization

Purpose
- Store reproducible visualization notebooks, static images, and dashboard configs that communicate project findings.

Suggested contents
- notebooks/ — Jupyter or Observable notebooks that generate figures
- figures/ — exported images (.png, .svg) with captions and provenance
- dashboards/ — Streamlit / Dash / Vega/Plotly configs (if used)
- README entries describing each key figure and how to regenerate it

Best practices
- Include the code, a short caption that explains the chart, and the exact command or notebook cell required to regenerate the figure.
